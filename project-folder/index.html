ข้อมูลหลักสูตร

ชื่อหลักสูตร: หลักสูตรวิศวกรรมศาสตรบัณฑิต

รหัสหลักสูตร: 25620151100293

ระดับการศึกษา: ปริญญาตรี

คณะ / มหาวิทยาลัย: คณะเทคโนโลยีและการจัดการอุตสาหกรรม/ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ

คำอธิบาย: พัฒนาคน เพื่อพัฒนาวิศวกรสารสนเทศและเครือข่าย ซึ่งจะเน้นการปฏิบัติเพื่อเป็นบัณฑิตที่คิดเป็น ทําเป็นที่สอดคล้องกับอัตลักษณ์ของมหาวิทยาลัยและคณะ
